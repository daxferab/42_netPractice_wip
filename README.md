# Description
---
The **Net Practice** project focuses on learning how networking works.
Consisting on 10 levels, it presents students with incorrectly configured network diagrams that they must fix.
Once a level has been completed, a configuration file can be exported. The student must submit 10 configuration files corresponding to each exercise.
During the evaluation, the student has to fix 3 random network diagrams in 15 minutes or less.
